# Tool Paths

Expected local DayZ tool paths:

- DayZ Tools: `C:\Program Files (x86)\Steam\steamapps\common\DayZ Tools`
- Terrain Builder: `C:\Program Files (x86)\Steam\steamapps\common\DayZ Tools\Bin\TerrainBuilder\terrainBuilder.exe`
- Addon Builder: `C:\Program Files (x86)\Steam\steamapps\common\DayZ Tools\Bin\AddonBuilder\AddonBuilder.exe`
- WorkDrive: `C:\Program Files (x86)\Steam\steamapps\common\DayZ Tools\Bin\WorkDrive\WorkDrive.exe`
- DayZ Server: `C:\Program Files (x86)\Steam\steamapps\common\DayZServer`
