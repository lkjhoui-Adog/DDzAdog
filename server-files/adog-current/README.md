# Adog Current Server Snapshot

This folder captures the small Adog-specific files that changed during the Michigan Survival server work.

Included:
- serverDZ.cfg from the Adog server folder
- desktop launch shortcut command file
- DayZ Launcher local mod registration and default preset files
- runtime-state.json with the current server process command line and intended mod order
- custom-file-hashes.json for custom Michigan files
- patched-third-party-server-file-hashes.json for third-party PBOs resynced from the local client install
- latest-server-evidence.txt with RPT evidence showing the MichiganSurvival world/mod loading

Not included:
- full DayZ server install
- full Workshop mod folders
- full profile logs and dumps
- third-party mod payload backups
